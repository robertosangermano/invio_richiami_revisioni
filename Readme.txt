questo script permette di inviare  email simultanee caricando un file csv , selezionando  l'intervallo della data in allegato trovate un video  del funzionamento. Questo codice non richiede api di software di email-marketing  come ad esempio Brevo. Unica email accettata email di tuo dominio come ad esempio : casa@tuodominio.it .
